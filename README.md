*TaskFlow – Vue 3 + Vue Router + Vite

TaskFlow é um projeto de gerenciamento de tarefas desenvolvido com foco em consolidar conhecimentos no ecossistema Vue 3.

*Tecnologias utilizadas

Vue 3 (Composition API)

Vue Router v4

Vite

JavaScript (ES6+)

LocalStorage para persistência de dados

*Funcionalidades

Criação e remoção de tarefas

Marcação de tarefas como concluídas

Filtros (Todas | Pendentes | Concluídas)

Navegação por rotas

Componentização e organização de views

*Objetivo

Demonstrar experiência prática com Vue 3 e seu ecossistema, aplicando boas práticas de estruturação, reatividade e gerenciamento de rotas em um projeto real.
